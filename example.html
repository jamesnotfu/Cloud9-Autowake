Hello, I'm Autowake


Autowake is an HTML file.

Servers hosted with the cloud-based IDE service Cloud9 go to hibernation every couple of hours, and when loading a URL hosted in Cloud9, it will not load the files correctly. This is an auto-invoking script that wakes the Cloud9 server up BEFORE loading the URL

FAQ.

How does it work?
Autowake is similar to a domain-forwarding/domain-masking page. It masks the current URL, and while doing this it waits until everything is loaded n' ready before loading the files.
